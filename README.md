# SpotyAPI

## Description

This project is a simple API that allows you to search for artists, albums and tracks using the Spotify API.

## Installation

Using PDM:

```bash
pdm install
```

## CLI
    
```bash
spoty --query <id> --type <type> --limit <limit> --features <features>
```
